Zac and Scott's Ruby Cheat Sheet

Hashes

Description

A Hash is a dictionary-like collection of unique keys and their values. Also called associative arrays, they are similar to Arrays, but where an Array uses integers as its index, a Hash allows you to use any object type.

Methods

  • #include?: Returns true if the given key is present in hash.
  • #length: Returns the number of key-value pairs in the hash.
  • #shift: Removes a key-value pair from hash and returns it as the two-item array [ key, value ], or the hashâ€™s default value if the hash is empty.
  • #to_a: Converts hash to a nested array of [ key, value ] arrays.

Arrays

Description

Arrays are ordered, integer-indexed collections of any object. Array indexing starts at 0, as in C or Java. A negative index is assumed to be relative to the end of the array---that is, an index of -1 indicates the last element of the array, -2 is the next to last element in the array, and so on.

Methods

  • #concat(other_array): Appends the elements of other_array to self.
  • #each {|item| block}: Calls the given block once for each element in self, passing that element as a parameter.
  • #sample: Choose a random element from the array
  • #empty?: Returns true if the array contains no elements

Strings

Description

A String object holds and manipulates an arbitrary sequence of bytes, typically representing characters. An example is "Hello World!"

Methods

  • #length: Returns the character length of the string
  • #downcase: Returns a copy of the string with all uppercase letters replaced with their lowercase counterparts
  • #chars: Returns an array of characters in the string
  • #capitalize: Modifies the string by converting the first character to uppercase and the remainder to lowercase

Integers

Description

Holds Integer values that can be represented in a native machine word. Examples being 1 or 10.

Methods

  • #even?: Returns true if int is an even number.
  • #next: Returns the Integer equal to int + 1
  • #to_s: Returns a string containing the representation of the integer
  • #size: Returns the number of bytes in the machine representation of the integer
